## Getting Started

### Prerequisites
- Python 3.8 or higher
- Telegram Bot API token
- Required dependencies (see `requirements.txt`)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yakulevichvlad/coursework-telegram-bot.git
cd coursework-telegram-bot
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Configure your bot:
   - Set up your Telegram Bot API token
   - Configure any necessary API keys for AI services

4. Run the bot:
```bash
python main.py
```

## Usage

Start a conversation with the bot on Telegram and:

1. **Choose a Topic**: Specify your coursework topic
2. **Configure Settings**: Set preferences for chapter count, content depth, citation style
3. **Generate Content**: Let the AI create structured content
4. **Review & Edit**: Analyze the generated content and make adjustments
5. **Export**: Download your completed coursework

## Commands

- `/start` - Initialize the bot and display main menu
- `/help` - Show help information
- `/newtopic` - Start a new coursework topic
- `/settings` - Configure bot preferences
- `/generate` - Generate coursework content
- `/analyze` - Run AI analysis on current content
- `/export` - Download your coursework
